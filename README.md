🚀 Getting Started
# Clone the repository
git clone https://github.com/Alborithm/FloorDigitalization.git

# Restore dependencies
dotnet restore

# Build the project
dotnet build
▶️ Running the Application

🧪 Running Tests
dotnet test - TBD
🏗️ Build for Release
dotnet publish -c Release -r win-x64 --self-contained true

You can change the runtime identifier (-r) for different platforms:

win-x64
osx-x64
linux-x64
